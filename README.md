# Code Pathway - AI Coding Learning Platform

## Overview
Code Pathway is an AI-powered coding learning platform that provides AI-driven personalized feedback on user-submitted code. The platform is designed to improve learning outcomes by offering real-time insights into code quality, structure, and efficiency.

## Features
- Code submission and evaluation
- AI-generated personalized feedback
- Interactive lesson modules
- User progress tracking
- Scalable deployment using AWS EC2

## Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- AWS EC2
- Database (for storing user data and submissions)

## System Workflow
1. User submits code through the web interface  
2. Backend processes the input  
3. AI generates feedback  
4. Feedback is stored and returned to the user
